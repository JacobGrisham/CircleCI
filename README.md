# robofriends-testing

[![JacobGrisham](https://circleci.com/gh/JacobGrisham/CircleCI.svg?style=svg)](https://app.circleci.com/pipelines/github/JacobGrisham/CircleCI)

To run the project:

1. Clone this repo
2. Run `npm install`
3. Run `npm start`
4. Run tests with `npm test` -- you may have to press "a" to run all tests
